# Dodgefall: Neon Survival

A fast-paced **dodge-the-falling-blocks** survival game with a neon vibe.  
Stay alive as the sky turns hostile—**weave through chaos, dash out of danger, build momentum, and stack rewards** the longer you last.

## What it is
**Dodgefall: Neon Survival** drops you into an arena where blocks rain down in unpredictable patterns.  
Your goal is simple: **survive, score big, and keep improving** with unlocks and progression.

## Highlights (no spoilers)
- Smooth movement + a clutch **dash** for escapes
- **Advancements** to chase as you improve
- **Upgrades** and progression that keep runs feeling fresh
- Pickups and surprises that change how each run plays
- A clean neon UI with satisfying effects

## Controls
- Move: **WASD** / **Arrow Keys**
- Dash: **Space**
- Pause: **P**
- Restart: **R**
- Menu/Back: **Esc**

## How to play
1. Download or copy the game into a file named `dodge_blocks.html`
2. Open it in your browser (Chrome/Safari/Edge)
3. Hit **Play** and survive as long as you can

## Notes
This is designed to be lightweight and run directly in a browser — no installs needed.

---
Good luck. The blocks don’t care.
